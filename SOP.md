# 📅 每日公众号发文 SOP (wx_articles v4.3) — 精简实战版

## 1. 早安：选题雷达 (09:00)
1. 启动后台：`streamlit run app.py`
2. 打开 **Topic Radar** 标签页。
3. 输入今日核心意图（可选，如 "DeepSeek"），点击 **🚀 Start Scan**。
4. 浏览扫描结果，看到心动的选题，点击 **Select**（此时已存入历史，防止撞车）。

## 2. 午休：自动生产 (12:00)
*（在 Streamlit 侧边栏操作）*
1. 点击 **🏆 生成决策 (Final Decision)**：综合早上的扫描，定下唯一选题。
2. 点击 **🔬 开始研究 (Start Research)**：AI 自动去搜集资料、看代码、写笔记。
3. 点击 **✍️ 生成初稿 (Generate Draft)**：AI 基于笔记写出第一版文章。

## 3. 下班前：润色与发布 (17:30)
1. 切换到 **📝 Editor** 标签页，选择 `Draft (draft.md)`。
2. **审查与润色**：
   - 检查文章逻辑。
   - 输入修改意见（如“把开头改得更有悬念”），点击 **Refine**。
   - **补充截图**：看到文中 `> TODO: [截图]`，手动截图放入 `5_assets`，或检查自动截图是否成功。
3. **事实核查**：点击 **🕵️ Run Audit**，确认没有瞎编乱造。
4. **排版发布**：
   - 右侧预览区选择 `Livid` (极客) 或 `Vue` (清新) 风格。
   - 点击预览区，`Ctrl+A` 全选 -> 复制。
   - 粘贴到微信公众号后台，**手动上传图片**替换占位符。
5. **发送！** 🚀
